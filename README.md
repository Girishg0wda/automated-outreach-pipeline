# Automated Outreach Pipeline

A Python-based automated cold outreach system that takes a company domain as input and performs:

1. Company Discovery
2. Contact Discovery
3. Email Resolution
4. Automated Outreach

## Features

* Modular architecture
* Environment variable configuration
* CSV export
* Logging
* Safety confirmation before sending emails
* Extensible API integrations

## Project Structure

```text
automated-outreach-pipeline/
├── main.py
├── config.py
├── services/
├── utils/
├── outputs/
└── README.md
```

## Installation

